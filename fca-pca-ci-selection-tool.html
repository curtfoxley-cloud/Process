Configuration Audit Process · Appendix C · Decision Tool

FCA/PCA
CI Selection

Determine whether a Configuration Item should be designated for Functional and/or Physical Configuration Audit per the Configuration Audit Process (FCA/PCA), Appendix C.

When to run this assessment — §5.2.1 / Appendix C

Complete one assessment per CI at initial audit planning, at each CM-plan revision, and on any Part D trigger event: substantive design, process, facility, tooling, or supplier change (SM-0001 MoC); new quality escape or safety event traceable to the CI; CSA reconciliation anomaly; hazard log revision adding a risk control to the CI. Record the result and rationale for every CI evaluated — designated or not — and attach it to the Audit Designation Assessment Record (Section 6).

Reference: Configuration Audit Process (FCA/PCA), §5.2.1 Audit Designation and Appendix C — FCA/PCA CI Selection Checklist. Selection of the audit type (FCA, PCA, both, or delta audit of open scope) follows the contract scope and program judgment; this tool determines whether designation is required. It is a decision aid only and does not substitute for engineering judgment, System Safety concurrence, or contractual requirements.
